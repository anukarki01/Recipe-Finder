A simple and responsive Recipe Finder web application built using React.js.  
Users can enter ingredients to find recipes with instructions, images, and nutritional information. The app uses the free TheMealDB API.

** Features**


- Search recipes by ingredients or meal name
- View detailed recipe information (ingredients, instructions, category)
- Save favorite recipes using localStorage
- Filter and sort recipes
- Responsive and user-friendly design
- Handles loading, errors, and empty results
